## Angle
Use the GitHub Copilot CLI command center story as the setup, then pivot to enterprise reality: personal productivity prototypes move fast, but organization-wide deployment fails without governance controls.

## Why this fills a gap
Existing governance post (000003) defines a control model. This post applies that model to a concrete, current Copilot CLI build pattern and shows what must change when moving from one builder to one hundred builders.
