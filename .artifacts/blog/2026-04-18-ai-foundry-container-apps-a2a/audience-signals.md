# Audience Signals: AI Foundry Agent-to-Agent Architecture
## Audience Problem
Teams building with Azure AI Foundry want agent-to-agent communication patterns (A2A). Foundry supports agents calling other agents, but there's no clear reference architecture. Teams are building custom routing layers or using ad-hoc message passing.
## Audience Alignment
- AI engineers designing Foundry agent systems
- Platform teams building agent orchestration
- Solution architects on complex AI projects
