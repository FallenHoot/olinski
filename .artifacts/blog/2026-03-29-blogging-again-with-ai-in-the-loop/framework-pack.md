# Framework Pack

## Framework name

Human-Gated AI Writing Loop

## Decision framework

Use this when deciding whether an AI-assisted draft is publish-ready.

1. Signal quality
   - Is the topic current and relevant to the audience?
2. Thesis clarity
   - Is there one clear decision or position the post supports?
3. Evidence sufficiency
   - Are non-obvious claims tied to public sources?
4. Risk control
   - Has fact-risk and confidentiality screening been completed?
5. Human accountability
   - Has a human reviewer approved final framing and claims?

## Red flags

- Claim sounds precise but has no source.
- Language implies official Microsoft announcements.
- Generic advice without concrete action.
- Trend already saturated with no differentiated take.

## Weekly checklist

- Pick one priority topic from backlog.
- Run Microsoft and watchlist signal scans.
- Build a short evidence table before drafting.
- Run contrarian and fact-risk review.
- Publish only after explicit human approval.
