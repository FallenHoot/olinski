# Audience Signals: FinOps with Azure Data Explorer

## Audience Problem
FinOps teams lack visibility into cloud cost drivers. Azure Data Explorer (ADX) can ingest cost data and create dashboards, but many teams don't know ADX exists or how to set it up. Teams are using spreadsheets, BI tools (Power BI), or Databricks for cost analysis—all requiring manual data pipelines and lack real-time cost trending.

## Key Signals
- ADX pricing is coming down (Azure blog announcements)
- FinOps practitioners discovering ADX for cost analytics (conference talks)
- Microsoft publishing cost analysis templates using ADX
- Growing adoption of Azure Cost Management exports to ADX

## Audience Alignment
- FinOps engineers and managers
- Cost center owners needing real-time visibility
- Platform teams building cost dashboards
