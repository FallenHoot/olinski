# Framework Pack: Sovereignty Decision

## Compliance Needs Matrix
| Need | Solution | Cost | Maintenance |
|------|----------|------|-------------|
| Audit rights | Global cloud + contract | Low | Low |
| Data residency | Regional deployment | Low-Med | Low |
| Full sovereignty | Separate cloud | High | Very High |

## Recommendation Flow
1. Do regulations require sovereignty? → Only China/Russia
2. Does GDPR apply? → Use EU regions + audit rider instead
3. Does UK/Canada apply? → Use regional deployments
4. Default: Global cloud + audit rights contract

## Action Checklist
1. Audit compliance requirements (not politics)
2. Check if regulation actually requires sovereign cloud
3. Evaluate: audit rights + residency vs full sovereignty
4. Make decision based on cost/compliance analysis (not vendor marketing)
