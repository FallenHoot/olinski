# Framework Pack: Squad Implementation

## Squad Core Concepts
1. **Small teams** (5-8 people per squad)
2. **Clear ownership** (one person leads decisions per initiative)
3. **Async updates** (written status, not meetings)
4. **Visible progress** (shared dashboard or GitHub wiki)
5. **Low ceremony** (documents in Markdown, versioned in Git)

## Squad Setup Template
1. Create `/squads/` folder in your repo
2. Each squad has:
   - `README.md` - team charter and contacts
   - `status.md` - weekly progress (updated every Friday)
   - `decision-log.md` - key decisions and rationale
3. Roll up status to company dashboard monthly
