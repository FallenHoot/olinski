# Framework Pack: AI CoE Decision
## CoE vs Platform Model Decision Matrix
| Factor | CoE Better | Platform Better |
|--------|-----------|-----------------|
| Team size | <50 | >50 |
| Regulatory | High | Low |
| Adoption pace | Slow learners | Fast movers |
| Centralization | Necessary | Preferred local |

## Action Checklist
1. Assess governance needs (regulation, compliance, risk)
2. Count teams using AI (if <20, reconsider strategy)
3. Evaluate platforms vs CoE cost
4. Start with lightweight model (review gate only)
