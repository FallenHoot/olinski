# Framework Pack

## Framework name

PTU vs TPM Decision Tree

## Decision process

1. Do you have 12 months of production token history?
   - No → Use TPM
   - Yes → Continue

2. Is token volume consistent within 20% month-to-month?
   - No → Use TPM
   - Yes → Continue

3. Is runaway cost risk higher than PTU savings?
   - Yes → Use TPM
   - No → Continue

4. Can procurement process handle quarterly PTU reviews?
   - No → Use TPM
   - Yes → Use PTU if utilization > 60%
