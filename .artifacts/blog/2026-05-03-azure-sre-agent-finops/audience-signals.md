# Audience Signals: Azure SRE with Agent-Based FinOps
## Audience Problem
SRE teams troubleshooting cost anomalies use ad-hoc investigation (manual queries, dashboards). AI agents can automate anomaly investigation—correlate cost spikes with deployments, config changes, traffic patterns. Teams don't yet have patterns for SRE + AI agent workflows.
## Audience Alignment
- SRE teams managing costs
- FinOps engineers
- Platform teams automating operations
